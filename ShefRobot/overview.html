ShefRobot

This package allows Java to communicate over bluetooth with Lego EV3 robots.

Binaries

You can download a compiled .jar of this package to add to your classpath here.

You can download an archive with ShefRobot.jar and it's dependencies here.

Sample Code

A small number of examples can be found here.

Download the desired .java file, place these in the same directory as the files contained within ShefRobot.zip. Open cmd (terminal) and navigate to the directory. Run setupConsole.bat (setupConsole.sh), this will configure your classpath for the lifetime of the cmd (terminal) window. Now you can compile the desired Java file as you would normally.
